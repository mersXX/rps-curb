# Rock-Paper-Scissors game

## Prerequisites
* ruby 2.5.0
* rails 5.2.1

## Setup
Run in the terminal:

```bash
$ bundle
$ rake db:setup
$ rails s
```
## Rules
Select one of the options and click on it, the alert will report the result (you can see the full details in response)
* Rock beats scissors
* Scissors beats paper
* Paper beats rock
* Identical throws draw
